# Agentic AI HR Assistant (RAG-Based System)

## Overview
This project implements an agentic AI HR assistant that answers HR-related questions using retrieval-augmented generation (RAG). The system retrieves relevant policy sections from internal documents and generates grounded, explainable responses.
